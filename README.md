# 📺 Aaron's Portfolio

My portfolio, broadcast on a vintage CRT television. Every channel is one of
my projects — flip through them with the channel buttons on the set (or the
`←` `→` arrow keys). Some channels play a teaser of the project on a loop;
press `TELETEXT` for the full write-up and links. Yes, the power knob works.
Yes, there is static.

And if you turn the TV off… you might find out where you've been sitting
this whole time. (WASD to walk, mouse to look, click the TV to sit back down.)

https://aarkro.github.io/portfolio/

## Stack

React · TypeScript · Vite · SCSS
